# Yemek / Kalori Takip Uygulaması

Bu proje, Android Studio ortamında Java kullanılarak geliştirilmiş basit ve işlevsel bir yemek/kalori takip uygulamasıdır.  
Uygulama, kullanıcıların günlük tükettikleri yemekleri ve kalori bilgilerini tarih bazlı olarak kaydetmelerini ve takip etmelerini sağlar.

## Özellikler
- Tarih seçimi (DatePickerDialog)
- Öğün seçimi (Spinner)
- Kalori girişi
- SQLite veritabanı kullanımı
- Kayıt ekleme, listeleme, güncelleme ve silme (CRUD)
- Günlük toplam kalori hesaplama

## Kullanılan Teknolojiler
- Android Studio
- Java
- SQLite
